# drone-delivery-system
Description: Simple Java project to practice programming logic.  Drone delivery system that verifies: - if the destination is within the permitted area (10 ≤ X ≤ 50, 10 ≤ Y ≤ 80) - if the cargo weight is valid (up to 10% of the drone's weight) - if the battery is sufficient (long distances: minimum 50%, short distances: minimum 20%)
